# Eloy Dotfiles

## Tools:

  - Vim/Neovim
  - Tmux
  - Alacrity
  - Ack
  - Gitconfig
  - jq

## Installation

This repo is using [Chezmoi](https://github.com/twpayne/chezmoi) to manage
dotfiles

```
git@github.com:eloycoto/dotfiles.git
cd dotfiles
make help
make install
```


## Add files

```
make chezmoi add ~/config_file
```
